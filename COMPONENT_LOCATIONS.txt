================================================================================
DATA FLOW EXAMPLES
================================================================================

LEAD MANAGEMENT FLOW:
User selects date range → LeadCRMTab fetches /api/crm/leads
→ Displays table with filtering → User clicks row
→ LeadDetailModal opens, fetches /api/crm/lead/:leadId and /api/crm/lead/:leadId/calls
→ Displays modal with status dropdown
→ User changes status → Clicks Save
→ Calls PUT /api/crm/lead/:leadId with new status
→ Modal updates local state with response

RETENTION FLOW:
User selects date range → RetentionDashboardTab fetches /api/crm/policyholders
→ Displays table → User clicks row
→ PolicyholderDetailModal opens, fetches /api/crm/policyholder/:policyNumber and /api/crm/tasks
→ Displays modal with forms
→ User logs outreach → Calls POST /api/crm/tasks
→ Outreach history updates in modal

BUSINESS HEALTH FLOW:
User selects date range → BusinessHealthTab fetches /api/crm/metrics/business-health
→ Displays KPI tiles, breakdown charts, trends, and carrier table
→ All data derived from single endpoint response
→ User can click carrier row for drill-down view
